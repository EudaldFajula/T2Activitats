# Exercici18

### Classes d'equivalencia

Numeros entre el 0 i el 120.

Numeros fora del rang de 0 i 120.

### Casos extrems (Valors limits)

- -1
- 0 
- 1
- 119
- 120
- 121

Perquè estan en el limit
